Projeto: Biblioteca

Instalar o Laravel Jetstream.
Permitir o login com 2FA.
Customizar o tema com componentes https://daisyui.com/components/
Criar um repositório no Github com a conta da Inovcorp.



Teremos 3 menus, onde teremos em cada um deles tabelas:

Livros
Autores
Editoras

Nestas tabelas deve ser possível pesquisar, ordenar e filtrar pelos campos.

Livros

Nos Livros teremos os seguintes campos:

ISBN
Nome
Editora (relação)
Autores (relação e pode ser mais que 1)
Bibliografia
Imagem da Capa
Preço


Autores

Nos Autores teremos os seguintes campos:
Nome
Foto


Editoras

Nas Editoras teremos os seguintes campos:
Nome
Logótipo
Notas

Ter os dados cifrados na base de dados.
Capacidade de exportação de ficheiro excel com os dados da base de dados de Livros.
